◆ get_numerical_question_form_data_pi()

qtype_numerical_test_helper::get_numerical_question_form_data_pi ( )

Get the form data that corresponds to saving a numerical question.

This question asks for Pi to two decimal places. It has feedback for various wrong responses. There is hint data there, but it is all blank, so no hints are created if this question is saved.

Return values
stdClasssimulated question form data.

Get the form data that corresponds to saving a numerical question.

Like get_numerical_question_form_data_pi(), but this time with two hints, making this suitable for use with the Interactive with multiple tries behaviour.
