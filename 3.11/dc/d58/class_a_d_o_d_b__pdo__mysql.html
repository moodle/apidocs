Quotes a string to be sent to the database.

If we have an active connection, delegates quoting to the underlying PDO object PDO::quote(). Otherwise, replace "'" by the value of $replaceQuote (same behavior as mysqli driver).

Parameters

bool$magic_quotesThis param is not used since 5.21.0. It remains for backwards compatibility.
