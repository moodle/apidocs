Detailed Description

Interface to indicate that a question bank column can go in the action menu.

If a question bank column implements this interface, and if the edit_menu_column is present in the question bank view, then the 'column' will be shown as an entry in the edit menu instead of as a separate column.

Probably most columns that want to implement this will be subclasses of action_column_base, and most such columns should probably implement this interface.

If your column is a simple action, you can probably save duplicated code by using the base class action_column_menuable as an easy way to implement both action_column_base and this interface.
