◆ i_drag_to_space_in_the_drag_and_drop_into_text_question()

behat_qtype_ordering::i_drag_to_space_in_the_drag_and_drop_into_text_question ( string $label,
int $position )

Drag the drag item with the given text to the given space.

Warning, only works if the question is using a behaviour like Immediate feedback that has a check button.

Also, do not use this to drag an item to the last place. Just drag all the other non-last items to their place.

Parameters
string$labelthe text of the item to drag.
int$positionthe number of the position to drop it at.

@Given /^I drag "(?P<label>[^"]*)" to space "(?P<position>::d+)" in the ordering question$/
