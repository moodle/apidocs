Convert page names to URLs for steps like 'When I am on the "[identifier]" "core_form > [page type]" page'.

Recognised page names are: | pagetype | name meaning | description | | Fixture | script-name | Fixture file name without extension |

The fixture name should be the filename without path or extension. E.g. autocomplete-disabledif for lib/form/tests/fixtures/autocomplete-disabledif.php.

Parameters
string$typeidentifies which type of page this is, e.g. 'Fixture'.
string$identifieridentifies the particular page, e.g. 'autocomplete-disabledif'.
Return values
moodle_urlthe corresponding URL.
Exceptions
Exceptionwith a meaningful error message if the specified page cannot be found.
