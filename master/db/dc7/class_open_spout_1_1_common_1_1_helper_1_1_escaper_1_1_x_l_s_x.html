◆ escapeControlCharacters()

OpenSpout\Common\Helper\Escaper\XLSX::escapeControlCharacters (   $string)
protected

Converts PHP control characters from the given string to OpenXML escaped control characters.

Excel escapes control characters with xHHHH and also escapes any literal strings of that type by encoding the leading underscore. So "::0" -> x0000 and "_x0000_" -> x005F_x0000.

NOTE: the logic has been adapted from the XlsxWriter library (BSD License)
