OpenSpout\Writer\XLSX\Writer::mergeCells ( array  $range1,

Merge cells.

Row coordinates are indexed from 1, columns from 0 (A = 0), so a merge B2:G2 looks like $writer->mergeCells([1,2], [6, 2]);.

You may use CellHelper::getColumnLettersFromColumnIndex() to convert from "B2" to "[1,2]"

Parameters
int[]$range1- top left cell's coordinate [column, row]
