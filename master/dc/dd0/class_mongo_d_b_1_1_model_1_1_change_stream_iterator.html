MongoDB\Model\ChangeStreamIterator::getInnerIterator ( )
final

Necessary to let psalm know that we're always expecting a cursor as inner iterator.

This could be side-stepped due to the class not being final, but it's very much an invalid use-case. This method can be dropped in 2.0 once the class is final.

◆ getResumeToken()

MongoDB\Model\ChangeStreamIterator::getResumeToken ( )

Returns the resume token for the iterator's current position.

Null may be returned if no change documents have been iterated and the server did not include a postBatchResumeToken in its aggregate or getMore command response.
