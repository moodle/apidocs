Google_Service_CivicInfo_Divisions_Resource::search (   $optParams = array())

Searches for political divisions by their natural name or OCD ID.

(divisions.search)

Parameters
array$optParamsOptional parameters.

@opt_param string query The search query. Queries can cover any parts of a OCD ID or a human readable division name. All words given in the query are treated as required patterns. In addition to that, most query operators of the Apache Lucene library are supported. See http://lucene.apache.org/core/2_9_4/queryparsersyntax.html
